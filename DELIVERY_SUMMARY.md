- **Purpose:** Manage faculty members
- **Operations:** Add, edit, delete, view all faculty
- **Search:** By qualification, specialization, or ID
- **Status:** Track employment status
- **Database:** Synced with faculty table (7 fields)

### 3. Attendance Page (`/attendance`)
- **Purpose:** Mark and track student attendance
- **Operations:** Mark attendance, view records, summary
- **Analysis:** Attendance percentage, breakdown by status
- **Status:** PRESENT, ABSENT, LATE, EXCUSED (color-coded)
- **Database:** Synced with attendance table (6 fields)

### 4. Exams Page (`/exams`)
- **Purpose:** Manage exams and results
- **Operations:** Create exam, add results, publish
- **Types:** MIDTERM, FINAL, QUIZ, ASSIGNMENT
- **Grades:** A, B, C, D, F assignment
- **Status:** Draft or Published
- **Database:** Synced with exams and exam_results tables

### 5. Fees Page (`/fees`)
- **Purpose:** Track student fees and payments
- **Operations:** Create fee record, record payment, track status
- **Methods:** BANK_TRANSFER, CASH, CHEQUE, ONLINE, OTHER
- **Dashboard:** Total due, collected, and pending amounts
- **Status:** PENDING, PARTIAL, PAID, OVERDUE
- **Database:** Synced with fees and fee_payments tables

### 6. Users Page (`/users`)
- **Purpose:** Manage system users
- **Operations:** Create, edit, delete users
- **Fields:** Email, password, name, phone, role assignment
- **Status:** ACTIVE or INACTIVE
- **Statistics:** Total, active, and inactive user counts
- **Database:** Synced with users table (8 fields)
